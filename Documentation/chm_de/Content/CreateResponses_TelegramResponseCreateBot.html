Erweiterte Konfiguration - Einen Neuen Bot mit Telegram erstellen

Bevor Sie Kurznachrichten per Telegram empfangen können, müssen Sie die Telegram App auf Ihrem mobilen Endgerät mit Android, IOS oder Windows Phone Betriebssystem installieren. Desweiteren muss ein Bot erstellt werden. Der Bot ist ein Computerprogramm, der in der Telegram Kurznachrichten App wie ein normaler Nutzer dargestellt wird. Der Bot ist die Schnittstelle zwischen Ihrem Endgerät und dem EventMessenger.

Beginnen Sie mit der Installation der App aus dem "App Store" Ihres jeweiligen Endgerätes. In diesem Beispiel wird Android als Betriebbsystem verwendet. Die Installation auf einem IOS oder WindowsPhone Gerät verläuft analog dazu.

Rufen Sie nach erfolgter Installation die App auf.

Und Beginnen Sie die Konfiguration der App.

Telegram verknüpft ihr Telefon über dessen Rufnummer mit den Telegram Diensten. Dazu ist es nötig, das die App SMS und Anrufe empfangen kann.

Bitte gewähren Sie die obligatorischen Zugriffsrechte auf Ihr Gerät...

Bitte geben Sie Ihre Rufnummer ohne beginnende '0' ein. Aus "0172 123456" wird daher "172 123456"

Zur Authentifiezierung der Echtheit der angegebenen Daten erhalten Sie vom Telegram Dienst einen Code per Anruf oder SMS, der automatisch erkannt wird.

Nach erfolgreicher Konfiguration sehen Sie zunächst einen leeren Bildschirm. Sie können die Telegram App auch für Ihre herkömmliche Kommunikation mit den Kontakten aus Ihrer Kontaktliste verwenden. Wenn Sie dies wünschen, sollten Sie der App die notwendigen Rechte einräumen.

Ist die Konfiguratin der App abgeschlossen sehen Sie ersteinmal noch keine Einträge, da Sie noch keine Nachrichten empfangen haben.

Sie können nun entweder mit Kontakten aus Ihrer Kontaktliste eine neue Unterhaltung beginnen oder mit der Konfiguration des Bot fortfahren. Bitte rufen Sie in letzterem Fall den Browser auf Ihrem mobilen Endgerät auf. Die Konfiguration des Bots erfolgt mithilfe eines Telegram Dienstes der sich "Botfather" nennt.

Rufen Sie in Ihrem Browser bitte die Adresse "telegram.me/botfather" auf.

Sie werden auf die Startseite des Dienstes geleitet. Bitte nehmen Sie mit der Schaltfläche "Send Message" Verbindung mit dem Telegram Dienst auf.

Nach erfolgreicher Übernahme der Verbindungseinstellungen wird die Telegram App geöffnet und Sie sehen folgende Maske.
Klicken Sie bitte auf die Schaltfläche am unteren Bildschirmrand um mit dem Dienst zu kommunizieren und ihn anzuweisen, einen neuen Bot für Sie zu erstellen.

Der Dienst wird eine Anzahl an möglichen Befehlen anzeigen von denen der "/newbot" Befehl zur Erstellung eines neuen Bots gewählt wird. Der Dienst antwortet darauf mit der Nachfrage wie der neue Bot benannt werden soll.

Bitte vergeben Sie einen Namen für Ihren Bot.

Nach Vergabe einer Bezeichnung für Ihren Bot werden Sie gebeten einen Anmeldenamen zu vergeben. Dieser unterscheidet sich von der o.a. Bezeichnung dadurch, dass er auf "bot" endet.

Beispiel:

Bezeichnung: EventMessenger
Anmeldename: EventMessengerBot

Wenn Ihre Eingaben angenommen wurden, erhalten Sie einen Zugangsschlüssel (token).
Bitte kopieren Sie diesen Schlüssel auf eine geeignete Weise an einen sicheren Ort.

Sie können die Nachricht des Dienstes beispielsweise kopieren und in Ihr bevorzugtes E-Mailprogramm einfügen. Von dort aus können Sie Ihren Zugangsschlüssel leicht an den Rechner senden, auf dem Sie den EventMessenger installiert haben.

Haben Sie keine Möglichkeit hierzu, müssen Sie den Zugangsschlüssel bitte abtippen.

Bitte übertragen Sie den Zugangsschlüssel in den "Bot"-Konfigurationsdialog des EventMessengers.

Der EventMessenger ist nun bereit zur Versendung von Telegram Kurznachrichten. Bitte bewahren Sie den Zugangsschlüssel an einem sicheren Ort auf.
