Ereignisse anlegen - Neues Ereignis

 

  1. Bitte vergeben Sie für das neue Ereignis auf das Sie reagieren möchten einen eindeutigen Namen.
  2. Die Bezeichnung ist optional und erscheint im Hauptfenster als Hinweismeldung, wenn Sie auf Ihre angelegten Ereignisse mit der Maus zeigen.
  3. Wählen Sie einen Ereignistyp:
    1. "Transpondernutzungsereignis":
      Ein Transponder wurde an einer vernetzten Schließung benutzt.
    2. "Türüberwachungsereignis":
      Eine Person hat an einer "Door Monitoring" Schließung eine Aktion ausgelöst.
    3. "Eingangskanalereignis":
      Der Zustand an einem nicht potentialfreien Eingang eines WaveNet-, WaveNetManager-Routers oder LockNodes hat sich geändert.
  4. Klicken Sie auf "Ereignis konfigurieren" um die Bedingungen zu definieren unter denen das Ereignis von der LSM.Basic.Online abgefangen wird.
  5. Mit einem Klick auf "OK" wird das Ereignis angelegt.

Bitte beachten Sie, das sich Ereignisse nur dann anlegen lassen, wenn sie zuvor konfiguriert worden sind.
