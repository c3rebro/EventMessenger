Creating Responses - New KMTronic Device

 

  1. Please select a supported type of device.
  2. Please provide the IP address or hostname along with the TCP port that should be used to contact the Device.
  3. Continue to provide the login credentials.
  4. When finished, right click in a blank area within the "Relay Configuration" Group Box on the right.


  5. Open the Configuration Dialog for the "Web Relay" Device by selecting "Add new Relay Contact..."


  6. Select a Behavior for the relay contact:
    1. Turn On: Turn on the Relay Contact until it is turned off by a second response.
    2. Turn Off: Turn off a previos turned on relay contact.
    3. Pulse: Turns On a Relay Contact and automatically off after the specified duration.
    4. Toggle: Changes the relay state depending on its previos state: I.e. Off if it was On before; On if it was Off before.
  7. Click on "OK" to add this relay contact to the device configuration dialog.
  8. Repeat Steps 5 to 7 until finished.
  9. Click on "OK" in the Device configuration dialog to finish configuring the device.
