Reaktionen erstellen - Neue E-Mail

 

Um eine E-Mail versenden zu können, müssen die Empfängeradressen und eine Absenderadresse bekannt sein. Darüber hinaus werden die Daten des Servers benötigt, der die E-Mail versenden soll.

  1. Bitte geben Sie bei "Absender" die Absender - E-Mailadresse im Format user@emailserver.domain ein.
  2. Bitte verfahren Sie bei der Empfänger - E-Mailadresse analog. Sie können mehrere Empfängeradressen eingeben, wenn Sie diese mit einem Semikolon voneinander trennen.
  3. Wählen Sie einen Betreff.
  4. Vergeben Sie eine Nachricht die versendet werden soll. Die Nachricht kann Platzhalter enthalten die vor der Versendung der E-Mail ersetzt werden:

    Folgende Platzhalter sind möglich:
    '$lockname' = wird durch die Türbezeichnung ersetzt, an der die Transponderaktion statt fand
    '$lockphi' = ... durch die Seriennummer (PHI) der Schließung ...
    '$transpondername' = ... durch den Namen des Transponderbesitzers ...
    '$transponderphi' = ... durch die Seriennummer (PHI) des Transponders ...
    '$datetime' = ... durch den Zeitstempel ersetzt an dem das Ereignis auftrat
    '$accessresult' ... durch das Ergebnis des Zutrittversuchs / des Türüberwachungsereignisses ersetzt"

    Wird das Feld leer gelassen, wird eine Standardnachricht in deutscher Sprache versendet.

    Bitte beachten Sie, das nicht immer alle Platzhalter verfügbar sind. Beispiel: Ein Türüberwachungsereignis kennt die Platzhalter '$transpondername' und '$transponderphi' nicht. Unbekannte Platzhalter werden mit '$dont care' befüllt.
  5. Mit einem Klick auf "SMTP Einstellungen", können Sie die Maske zur Serverkonfiguration aufrufen, von dem aus die E-Mail versendet werden soll.


    Es ist nicht möglich eine E-Mail Reaktion ohne Servereinstellungen anzulegen.

    1. Geben Sie die URL des SMTP Servers im Format 'servername.domain' an.
    2. Definieren Sie den Port an dem der Server Nachrichten entgegennimmt.
    3. Wenn der Server eine Authentifizierung erfordert, setzen Sie bitte den Haken unter "Benutzername / Passwort verwenden" und geben Sie diese Daten ein.
    4. Mit einem Klick auf "OK" legen Sie Ihre Servereinstellungen fest.
  6. Haben Sie die SMTP Servereinstellungen angegeben, können Sie ihre E-Mail - Konfiguration mit einem Klick auf "OK" abschließen.

    Bitte vergessen Sie nicht Ihre konfigurierte E-Mail Ihrer Reaktionsliste hinzuzufügen.
