Reaktionen anlegen - Neue Reaktion

 

  1. Bitte vergeben Sie für die neue Reaktion einen eindeutigen Namen.
  2. Die Bezeichnung ist optional und erscheint im Hauptfenster als Hinweismeldung, wenn Sie auf Ihre angelegten Ereignisse mit der Maus zeigen.
  3. Wählen Sie einen Reaktionstyp:
    1. "Gerät: KNX":
      Ansteuerung von KNX Ger√§ten.
    2. "Gerät: KMTronic"   
      Ansteuerung von KMTronic WebRelais Komponenten.
    3. "Nachricht: E-Mail":
      Versendet E-Mails an einen oder mehrere Empfänger.
    4. "Nachricht: Telegram (AppStore)":
      Versendet Kurznachrichten an die Telegram App.
  4. Klicken Sie auf "Reaktion konfigurieren" um die Reaktionen im Detail zu konfigurieren.
  5. Mit einem Klick auf "Hinzufügen", wird die Reaktion der Liste abzuarbeitender Reaktionen hinzugefügt.
  6. Mit einem Klick auf "OK" wird die Reaktionsliste angelegt.

Bitte Beachten Sie, das sich Reaktionen nur dann hinzufügen lassen, wenn Sie zuvor konfiguriert worden sind.
