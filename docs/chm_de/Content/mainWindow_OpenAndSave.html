Hauptfenster - Speichern und Öffnen

 
Dies ist das Hauptfenster welches direkt nach dem Start erscheint.

  • Projekt öffnen (Tastenkombination: Alt + D, P):
    • Wählen Sie ein zuvor gespeichertes Projekt aus, um es wiederherzustellen.
  • "Speichern" (Tastenkombination: Alt + D, S)
    • Angelegte Ereignisse mit ihren dazugehörigen Reaktionen werden als Standarddatenbank gespeichert und können beim Start der Software automatisch geladen werden.
  • "Speichern unter..." (Tastenkombination: Alt + D, U)
    • Sie können auch mehrere Projekte anlegen. Wählen Sie dazu mit "Speichern unter..." einen Pfad und Dateinamen aus, unter dem die angelegten Ereignisse mit den dazugehörigen Reaktionen abgelegt werden sollen.
